# Linktree

Uma página simples de links pessoais, criada para centralizar site, blog, redes sociais e projetos em uma interface bonita, responsiva e direta.

![Preview do projeto](./src/assets/Screenshot.png)

## Stack

- Vite
- TypeScript
- Tailwind CSS
- ESLint
- Prettier

## Como rodar

Instale as dependências:

```bash
pnpm install
```

Inicie o servidor local:

```bash
pnpm dev
```

Gere o build de produção:

```bash
pnpm build
```

## Scripts úteis

```bash
pnpm lint
pnpm lint:fix
pnpm format
pnpm format:check
```
